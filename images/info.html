Joystick (control movement)

Hardware Output:

1 LEDs (Indicates use of joystick)

Description

Fast and Furious is a one player game with the simple goal of reaching the highest score. Score points are obtained by the collision of the red car and the coins that spawn on screen. There is also an obstacle that spawns on screen that is the image of a boulder. If the boulder is touched at any time the current game will be over. The car is looped on an image of a track/road to give it the illusion of the car moving. In order to move the car from left to right you must move the joystick from left to right. To connect the joystick an Arduino Joystick must be paired to the website. There is a connect option in the bottom left of the screen. Once the game is over the loop of the track is stopped and “Game Over” is displayed in blue words. To restart the game you must simply refresh the page. Each time the game is refreshed the joystick must be paired again.

Future Development

To improve the game I would implement more obstacles and a wider canvas so that the car has more room to dodge the obstacles. I would also implement more lives so that the user has more time to reach a higher score. As far as my controller I would like to include a restart button and also more LEDs to indicate lives left.
